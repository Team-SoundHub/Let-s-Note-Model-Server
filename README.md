# Let-s-Note-Model-Server
Let-s-Note 관련 모델 서버 입니다.
## Quick Start
### Set .env in root directory
```
OPENAI_API_KEY={write down ur api key}
```
### install venv
```
$ pip install virtualenv
$ virtualenv venv
$ source venv/bin/activate
```
### install packages
```
(venv) $ pip install -r requirements.txt
```
